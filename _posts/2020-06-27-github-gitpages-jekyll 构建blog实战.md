## github-gitpages-jekyll 构建blog实战



## 1 目标

完成一个自己的博客，写完后只需提交，即可触发自动编译部署。(简单-完全免费)



###### 博客的搭建过程

| 组件                    | 网址                       | 提供的功能       |
| ----------------------- | -------------------------- | ---------------- |
| github （仓库+action）  | https://github.com/        | action：持续集成 |
| gitpages（域名+服务器） | https://pages.github.com/  |                  |
| jekyll（网站模版）      | https://jekyllrb.com/docs/ |                  |
| 七牛云                  |                            | 辅助，存储图片   |
|                         |                            |                  |
