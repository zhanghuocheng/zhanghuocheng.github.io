#### 1 数组与切片的区别

| \    | 数组      | 切片     |
| ---- | --------- | -------- |
| 长度 | 固定      | 变长     |
| 类型 |           | 引用类型 |
| 长度 | 长度=容量 |          |
| 容量 | 长度=容量 | 扩容，   |

```
1 切片扩容，大于容量，先以2倍扩容，如果长度>=1024,则以1.25倍扩容。

```
