java VS go的区别

| 语法用例\语言                        | java                              | Go                   |
| ------------------------------------ | --------------------------------- | -------------------- |
| Switch/case 1 命中case,但是没有break | 会继续走下面的case,直到走完/break | 没有break也会结束    |
| 变量交换                             | 必须借用temp变量                  | 直接a,b=b,a 完成交换 |
| 循环依赖                             | 允许并解决                        | 不允许               |
